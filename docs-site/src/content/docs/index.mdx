## Two interfaces

nx-neptune provides two distinct ways to work with Neptune Analytics:

| | NetworkX Backend | Session Manager |
|---|---|---|
| **Use case** | Run algorithms on existing graphs | Project data lake sources into a graph |
| **API style** | Drop-in NetworkX replacement | Explicit lifecycle management |
| **Data source** | In-memory NetworkX graph | S3, Databricks, Snowflake, OpenSearch |
| **Setup** | Set `NETWORKX_GRAPH_ID` env var | Configure sources and IAM roles |
