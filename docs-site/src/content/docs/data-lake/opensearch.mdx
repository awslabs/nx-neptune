Project embeddings from **Amazon OpenSearch** indices into a Neptune Analytics graph.

## Overview

OpenSearch integration uses the Athena OpenSearch connector to query embeddings from OpenSearch indices and project them into Neptune Analytics, enabling combined graph analytics and vector similarity search.

## Demo notebook

The [OpenSearch demo notebook](https://github.com/awslabs/nx-neptune/blob/main/notebooks/import_open_search_embedding_demo.ipynb) demonstrates:

- Querying OpenSearch embeddings via Athena federated query
- Projecting embeddings into a graph
- Combining graph traversal with semantic similarity

## Use case: Product recommendation

Project product embeddings from OpenSearch into a graph. Use graph algorithms to find structurally related products, then refine with vector similarity for semantic relevance.

## Prerequisites

- Amazon OpenSearch domain with vector embeddings
- Athena OpenSearch connector deployed
- IAM permissions for Athena and the connector Lambda
